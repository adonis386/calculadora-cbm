# Sistema de Cálculo CBM y ft³

Sistema web para calcular metros cúbicos (CBM) y pies cúbicos (ft³) con conversión entre unidades.

## Características

- **Calculadora CBM**: Calcula metros cúbicos a partir de largo, ancho y alto en metros
- **Calculadora ft³**: Calcula pies cúbicos a partir de largo, ancho y alto en pies
- **Conversor de Unidades**: Convierte entre CBM y ft³

## Uso

1. Abra el archivo `index.html` en su navegador web
2. Ingrese las dimensiones en las calculadoras correspondientes
3. Haga clic en el botón "Calcular" o presione Enter
4. Use el conversor para cambiar entre unidades

## Fórmulas

- **ft³** = (alto × largo × ancho) / 28320
- **CBM** = ft³ / 35.31
- **1 CBM** = 35.31 ft³

## Archivos

- `index.html` - Estructura HTML de la aplicación
- `styles.css` - Estilos y diseño responsive
- `script.js` - Lógica de cálculo y conversión
